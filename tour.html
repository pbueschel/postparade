How parsing works
Same pipeline runs nightly across every condition book on your circuit.
1
Ingest PDF
Pull condition books from each track's host (Equibase / Brisnet feeds).
2
LLM extract
Claude parses each race into a structured schema (≈45 fields).
3
Rule validate
Cross-check against jockey club rules + last year's conditions for regression.
4
Match horses
Run every active horse against every parsed condition, rank by fit score.
